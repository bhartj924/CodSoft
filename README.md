# CodSoft
CODSOFT Python Internship Tasks

This repository contains the Python projects completed as part of the CODSOFT Internship.
Each task is created using Python and focuses on basic concepts, logic building, and GUI development.

Task 1 – To-Do List Application

A simple task manager where users can add, update, delete, and mark tasks as completed.
A modern GUI version is included.

Task 2 – Calculator

A basic calculator that performs arithmetic operations like addition, subtraction, multiplication, and division.
Includes a modern GUI design.

Task 3 – Password Generator

A program that generates strong random passwords.
Users can choose password length, strength level, and generate multiple passwords.
Modern GUI version included.

Task 4 – Rock-Paper-Scissors Game

A fun game where the user plays against the computer.
Displays choices, result, and tracks the score.
Built with an easy-to-use modern interface.

Task 5 – Contact Book

A simple contact management system.
Allows adding, viewing, searching, updating, and deleting contacts.
Modern GUI included.

**About**

These projects were created as part of the CODSOFT Python Programming Internship.
